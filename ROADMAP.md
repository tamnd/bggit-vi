## Guidelines

- Translate meaning, not words; prefer natural Vietnamese phrasing.
- Keep all code blocks, command lines, Git output, and identifiers in English.
- Technical terms: English first, Vietnamese gloss in parens on first occurrence per chapter.
- No em-dashes (`—`); use `---` (three hyphens) as in the source.
- Preserve all markup tags verbatim: `[i[...]i]`, `[fl[...]]`, `[flw[...]]`, etc.
- Preserve all explicit Pandoc anchors: `{#anchor-name}`.
- No AI-boilerplate headers in translated text.
